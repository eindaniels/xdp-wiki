# Origins Mod

Sollte klar sein, was das ist. Ihr könnt beim Betreten des Origins Servers, eine Origin auswählen die ihr für immer auf dem Server behalten wird, jede Origin hat andere Vorteile und Nachteile man sieht wie schwer eine Origin ist durch den Impact beim Wählen dieser ist von 1-3 bewertet, z.B. Avian ist low (1/3) und Phantom ist High (3/3). Hier gibts unten noch eine Übersicht der ganzen Origins im originalen;
